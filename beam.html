Beam Deflection Simulation



INTRODUCTION:

This is a simulator/calculator for beam deflections.
The "INPUT" section contains the controls for the simulation.
The "EQUATIONS" section contains the equations used generate the diagrams.


Controls:

  • - Beam Type: This drop down menu changes the type of beam to simulate/calculate.
  • - Load (F) (kN): This slider and text box change the value of applicable point loads.
  • - Distrubuted Load (w) (kN/m): This slider and text box change the value of applicable distributed loads.
  • - Moment (M0) (kN*m): This slider and text box change the value of applicable moments.
  • - Load Position (a) (m): This slider and text box change the position of applicable loads.
  • - Beam Length (L) (m): This slider and text box change the length of the beam.
  • - Moment of Inertia (I) (mm4): This slider and text box change the moment of inertia of the beam.
  • - Young's Modulus (E) (GPa): The slider and text box change the Young's modulus of the beam.
  • - Zoom Buttons: The button "Max Zoom" zooms in as much as possible on each diagram.
                             The button "Half Zoom" zooms in half as much as possible on each diagram.
                             The diagrams automatically zoom out until their entire plot is visible.
  • - Zoom for Free Body Diagram (%): This text box changes the zoom level for the free body diagram.
  • - Zoom for Deflection Diagram (%): This text box changes the zoom level for the deflection diagram.
  • - Zoom for Shear Diagram (%): This text box changes the zoom level for the shear diagram.
  • - Zoom for Moment Diagram (%): This text box changes the zoom level for the moment diagram.

EQUATIONS:

Below are the equations that are uses to generate the diagrams.
The equations change to match the selected beam type.






The equations are only valid if these assumptions are satisfied:

  • - The longitudinal axis does not experience any change in length.
  • - The cross-sections of the beam remain plane and perpendicular to the longitudinal axis.
  • - The cross-section is constant and retains its shape.
  • - The material is in the linear elastic range according to Hooke's Law.
  • - The deformed angles and displacements are small.
  • - The beam is made of a homogeneous material.
